# 口袋48成员回放下载器

## 使用方法
### Mac或Linux
1. 安装python及ffmpeg
2. 运行`pip install -r requirements.txt`
3. 运行`python review-downloader.py`

### Windows
[下载压缩包](https://wwa.lanzoui.com/b00n47cgd)解压后运行.exe文件
